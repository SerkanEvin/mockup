# Wall Art Visualizer

## Project info

A web application to visualize wall art in your space.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Navigate to the project directory
```sh
cd wall-art-visualizer-main
```

3. Install dependencies
```sh
npm install
```

4. Start the development server
```sh
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
